# Follow Bot -- Instagram

A simple Instagram follow bot for exploring automated marketing strategies. Built for learning purposes.

---

## Disclaimer

This project was built for educational purposes to explore Instagram's API behaviour and automation concepts. Using automated bots may violate Instagram's Terms of Service. Use at your own risk. This is not intended for production use.

---

## What is this?

A lightweight Python script that automates the Instagram follow action. Originally built as an experiment in social media marketing automation and Python scripting.

---

## Tech Stack

- Python

---

## Getting Started

```bash
git clone https://github.com/Arminderf-Singh/Follow-Bot-Instagram.git
cd Follow-Bot-Instagram
pip install -r requirements.txt

# Configure credentials
cp .env.example .env
# Add your Instagram credentials to .env

python bot.py
```

Test with a throwaway account. Instagram may temporarily lock accounts that show unusual activity.
